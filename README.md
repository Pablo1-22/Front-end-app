# Aplikacja Pogodowa (React + Redux)

Projekt zaliczeniowy na przedmiot Programowanie Frontend.

## Funkcjonalności:
- Wyświetlanie pogody dla wyszukanych miast.
- Prognoza 5-dniowa i szczegóły (wiatr, ciśnienie, opady).
- Ulubione miasta (zapamiętywane w przeglądarce).
- Globalna zmiana jednostek (C / F / K).

## Technologie:
- React
- Redux Toolkit
- React Router
- Axios (OpenWeatherMap API)
